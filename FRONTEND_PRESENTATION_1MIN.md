# Выступление: Frontend AI Finance Assistant

---

## Вариант 1: Технический акцент

Добрый день! Расскажу про фронтенд нашего приложения.

**Технология:** .NET MAUI — кроссплатформенный фреймворк от Microsoft. Одна кодовая база для Android, iOS, Windows и macOS.

**Архитектура:** MVVM-паттерн с Dependency Injection. Сервисы регистрируются через MauiProgram: DatabaseService и FinanceService как Singleton, страницы как Transient.

**UI-компоненты:** XAML для декларативной разметки. Кастомные графики через IDrawable: FinanceChartDrawable для линейных графиков, PieChartDrawable для круговых диаграмм, BarChartDrawable для столбчатых, RadarChartDrawable для радиальных.

**Страницы:** MainPage — главная с графиками и транзакциями, ChatPage — AI-чат для извлечения транзакций, StatisticsPage — статистика, AddTransactionPage — добавление транзакций, AIAssistantPage — AI-ассистент, ProfilePage — профиль пользователя.

**Интеграция с Backend:** FinanceService использует HttpClient для REST API. Автоматическое определение URL для Android эмулятора (10.0.2.2) и локального хоста. Таймауты 30 секунд, обработка ошибок с fallback-сообщениями.

**Особенности:** Локальное хранение настроек через Preferences API. Асинхронные операции везде — async/await для неблокирующего UI. Нативная производительность через компиляцию в нативный код.

**Результат:** Единый код для всех платформ, нативный UX на каждом устройстве, быстрая разработка и поддержка.

Спасибо!

## Вариант 2: Акцент на пользовательском опыте

Добрый день! Презентую фронтенд нашего приложения.

**Главное:** Кроссплатформенное приложение на .NET MAUI — одно приложение работает на Android, iOS, Windows и Mac. Пользователь получает одинаковый опыт на всех устройствах.

**Интерфейс:** Современный UI с кастомными графиками. Линейные графики показывают динамику баланса, круговые — распределение по категориям, столбчатые — сравнение периодов. Все графики адаптивные и интерактивные.

**Результат:** Красивый, быстрый, кроссплатформенный интерфейс, который работает одинаково хорошо на всех устройствах.
